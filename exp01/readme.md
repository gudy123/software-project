## 配置环境 & 初识gdal

​	1、下载vs2010，安装

​	2、使用codeblocks，建立工程，选择编译器为Microsoft visual c++ 2010，Release版本

![](http://ww1.sinaimg.cn/large/005Jzifvgy1fxpdt1o1hcj31120iyacg.jpg)
